SYNOPSIS
--------

Adds a "deparse" command to [`Devel::Trepan`](https://github.com/rocky/Perl-Devel-Trepan/wiki).

This can tell you exactly where you are stopped.

INSTALLATION
------------

To install this, run the following commands:

	perl Build.PL
	make
	make test
	[sudo] make install
